LINUX LINEA DE COMANDO PRACTICA:


 -man : es el manual de lo que uno quiera , ejemplo : man ls , man mkdir

 -ls (lista) : tiene distintos modificadores y sirve para listar cosas.
 
 -pwd : indica el directorio donde esta uno parado, tambien tiene modificadores.

 -cd : no tiene manual (man) pero si ingresamos help cd , nos muestra info 
       Permite acceder a los subdirectorios.
       
  cd .. : baja un nivel al subdirectorio anterior
  cd ../ se peuda bajar a mas profundidad utilizamos tab para saber donde estamos y a donde acceder.
